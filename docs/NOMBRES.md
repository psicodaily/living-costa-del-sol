# 🏷️ Biblia de nombres (ficticios estilo GTA)

> Para poder **vender** el juego, las **marcas y negocios reales** se sustituyen por
> nombres **ficticios que recuerdan al original pero son distintos** (como GTA hace
> con "Los Santos" en vez de "Los Ángeles"). Aquí guardamos los equivalentes.
>
> **Regla:** los **lugares reales** (Marbella, Puerto Banús, calles) se pueden usar;
> las **marcas/negocios** se inventan. Vamos rellenando esta tabla según aparezcan.

---

## Lugares / zonas → REALES ✅ (DECIDIDO)
> **Decisión:** los nombres de **lugares se mantienen REALES** (Marbella, Puerto
> Banús, Costa del Sol, las calles). Se pueden usar y dan el gancho de "es Marbella
> de verdad". Solo se inventan las **marcas/negocios** (abajo).

| Real | Se usa | 
|---|---|
| Marbella | **Marbella** (real) |
| Puerto Banús | **Puerto Banús** (real) |
| Costa del Sol | **Costa del Sol** (real) |
| Calles (Av. de Julio Iglesias, etc.) | **reales** |
